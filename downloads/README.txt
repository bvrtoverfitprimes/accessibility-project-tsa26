Accessibility Platform – Downloads

This folder contains small downloadable demo tools that can run on a user's computer.
These are for a school/demo project.

Magnifier Overlay (Python)
- File: magnifier_overlay.py
- Goal: show a small magnified view of the screen near the cursor.

How to run (Windows)
1) Install Python 3.10+ from https://python.org (check “Add Python to PATH”).
2) Open a terminal in this folder.
3) Install dependencies:
   pip install -r requirements.txt
4) Run:
   python magnifier_overlay.py

Controls
- Esc: quit
- + / - : zoom in/out

Notes
- This is a simple demo magnifier. It runs as a separate window on top of other windows.
- True "system-wide" captions and deep OS overlays usually require a desktop app or OS features.
