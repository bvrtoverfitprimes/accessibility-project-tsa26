Project Overview

This project demonstrates a path toward a broader accessibility media hub. Current delivered pieces include:

  • Downloadable live captions overlay (Windows demo) that transcribes system audio into an always-on-top captions window.
  • Downloadable magnification overlay (Windows demo) with lens and bar modes for reading and focus.
  • Playground web demos that show magnification and captions behavior in a controlled environment.
  • Admin user flow control for protected user management in the prototype.

Future expansion ideas (beyond the current prototype) include deeper personalization, device integration, and richer accessibility features. For the school project, we focused on demonstrating the core experience for captions and magnification.
